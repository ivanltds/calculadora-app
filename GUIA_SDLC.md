# Projeto Calculadora - Guia SDLC e Memória

Este documento serve como o guia mestre para o desenvolvimento do projeto da Calculadora. Ele segue um ciclo de vida de desenvolvimento de software (SDLC) simplificado e manterá um registro contínuo (memória) das ações e decisões tomadas.

## Fases do SDLC

### 1. Planejamento
**Objetivo**: Definir o escopo, requisitos e tecnologias que serão utilizadas.
- [x] Definir operações matemáticas básicas suportadas (soma, subtração, multiplicação, divisão).
- [x] Definir tecnologias a serem usadas (HTML/CSS/JS puro).
- [x] Esboçar a interface do usuário (design premium com glassmorphism).

### 2. Desenvolvimento
**Objetivo**: Escrever o código fonte para a calculadora.
- [x] Configuração da estrutura inicial de pastas e arquivos.
- [x] Construção da interface (HTML e CSS).
- [x] Implementação da lógica (JavaScript).
- [x] Integração da interface com a lógica e ajustes estéticos.

### 3. Teste
**Objetivo**: Garantir que a calculadora funciona corretamente em diferentes cenários.
- [x] Testes de operações básicas (1+1, 5*5, etc).
- [x] Testes de validação e casos extremos (divisão por zero, múltiplos pontos decimais, limite de dígitos).
- [x] Testes de usabilidade e responsividade (visualização em dispositivos móveis).

### 4. Deploy
**Objetivo**: Disponibilizar a calculadora para os usuários.
- [x] Preparação dos arquivos para produção.
- [x] Hospedagem do projeto em um serviço gratuito (GitHub).
